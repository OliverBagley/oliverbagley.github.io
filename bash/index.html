bash, cli and helper scripts for proxmox, lcxs, debian etc..

Cloudflare speedtest CLI with dependencies and setup

This script automates the installation and setup of the Cloudflare speedtest CLI tool, including all necessary dependencies, making it easy to run speed tests from the command line on systems like Proxmox, LCXS, or Debian.

bash -c "$(curl -fsSL https://raw.githubusercontent.com/OliverBagley/oliverbagley.github.io/master/bash/cloudspeed.sh)"

Credit for speed-cloudflare-cli goes to KNawm.

Auto configure for avahi and mdns

This script automates the installation and configuration of Avahi for mDNS, enabling service discovery on local networks for systems like Proxmox, LCXS, or Debian.

bash -c "$(curl -fsSL https://raw.githubusercontent.com/OliverBagley/oliverbagley.github.io/master/bash/avahi-mdns.sh)"
