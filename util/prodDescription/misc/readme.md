# Product Description Generator

This project is a simple tool to generate formatted HTML product descriptions based on a Handlebars template.

## Files

*   `index.html`: The single-page web application.
*   `description.html`: A Handlebars template for generating product descriptions. It includes placeholders for a title, description, a list of features, and a table of specifications.
*   `example.html`: An example of a populated HTML description generated from the template.

## Usage

Open the `index.html` file in your web browser to use the generator.

## Goal

The ultimate goal of this project is to create a single-page web application that allows users to:

1.  Enter a product title, description, a list of features, and a list of specifications.
2.  See a live, formatted preview of the generated HTML.
3.  Copy the generated plain HTML.
